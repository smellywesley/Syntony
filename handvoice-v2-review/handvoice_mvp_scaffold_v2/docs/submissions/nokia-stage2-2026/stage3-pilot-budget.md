# Proposed Stage 3 pilot and budget method

## Feasibility pilot

This is a proposed design, not a powered clinical trial. Confirm site capacity
before inserting participant counts.

| Element | Proposed method | Source/status |
|---|---|---|
| Setting | One community rehabilitation or ageing-care site | OPEN partner confirmation |
| Participants | Bottom-up: eligible weekly throughput × recruitment fraction × pilot weeks | OPEN site inputs |
| Operators | Staff who already support the selected workflow | OPEN interviews |
| Outcomes | completion, usable-capture rate, operator time, within-session repeatability where two valid repetitions are collected, annotated event agreement | Proposed |
| Safety | staff-assisted, seated where appropriate, stop on fatigue/discomfort, no automated clinical action | Requires professional review |
| Data | pseudonymous IDs, minimum retention, access control, deletion procedure | Requires institutional approval |

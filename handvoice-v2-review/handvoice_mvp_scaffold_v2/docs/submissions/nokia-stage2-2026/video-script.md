# Three-minute video script and storyboard

Use prerecorded deterministic footage. Do not depend on live network or a live
model call. Display “engineering-validation prototype — no diagnosis” throughout.

| Time | Picture/action | Narration purpose |
|---|---|---|
| 0:00–0:25 | Older adult with trained staff; current workflow sketch | State the validated problem and consequence; replace draft wording after interviews |
| 0:25–0:50 | Seven-step care flow | Show staff operator, older-adult performer and qualified human reviewer |
| 0:50–1:30 | T01, T02 and T03 prerecorded capture | Explain local AI/CV, same-clock audio and three short tasks |
| 1:30–2:05 | Inline timeline, then one deliberate low-quality capture | Deliver the wow moment; show deterministic reason and safe retry |
| 2:05–2:35 | Validation ladder and governance boundary | Engineering evidence now; older-adult feasibility only after approval |
| 2:35–3:00 | Proposed outcomes, sourced budget categories and close | State practical impact without disease or efficacy claims |

## Recording checklist

- Capture a known-good full run and a QC failure/retry run.
- Record the screen locally at 1080p; export H.264 MP4 plus a second offline copy.
- Keep narration under 390 spoken words and final runtime between 2:50 and 3:00.
- Add captions, high-contrast callouts and readable units.
- Never show an operator key, real participant identifier or personal media.
